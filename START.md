# Uvodni Setup za Typescript, PIXI.js i Git/GitHub

## ✅ 1. Naučiti TypeScript

TypeScript je nadogradnja na JavaScript koja uvodi statičku tipizaciju i bolju organizaciju koda, osobito za veće projekte.

### Obavezni materijali:
